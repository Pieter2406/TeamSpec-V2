# Changelog

All notable changes to the TeamSpec VS Code extension will be documented in this file.

## [0.1.0] - 2026-01-07

### Added
- Initial release of @teamspec chat participant
- Role-based slash commands (/ba, /fa, /arch, /dev, /qa, /sm, /des)
- Quick commands (/story, /feature, /adr, /bug, /sprint)
- Meta commands (/status, /context, /help)
- Workspace context detection (project, features, stories, ADRs)
- Agent prompt loading from workspace or bundled resources
- Quality gate enforcement
- Follow-up suggestions based on role
- Configuration settings for agent path, default role, gate enforcement
